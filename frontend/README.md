# Frontend

This folder contains the React + Vite client for What's My Ping.

## Scripts

- `npm run dev`: start the Vite development server
- `npm run build`: create a production build
- `npm run preview`: preview the production build locally
- `npm run lint`: run ESLint

## Development Notes

- The app entry point is [src/main.jsx](/Users/roshanadke/IdeaProjects/WhatsMyPing/frontend/src/main.jsx).
- The main UI lives in [src/App.jsx](/Users/roshanadke/IdeaProjects/WhatsMyPing/frontend/src/App.jsx).
- During local development, `/api` and `/health` are proxied to `http://localhost:4001` unless `VITE_API_BASE_URL` is set.

See the root README for the full project overview, setup steps, and API documentation.
